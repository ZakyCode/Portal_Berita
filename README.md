# 📰 Portal Berita - Laravel CRUD

Portal Berita adalah aplikasi sederhana berbasis web menggunakan **Laravel** untuk mengelola berita.  
Fitur utama:
- Tambah berita
- Lihat daftar berita
- Edit berita
- Hapus berita

---

## ⚙️ Teknologi
- PHP 8.x
- Laravel 11
- MySQL / MariaDB
- Bootstrap 5 (opsional untuk tampilan)
